Autostart on Login

Configure Clip Prompt to start automatically when you log in

macOS Instructions:

To enable autostart:

  1. Open System Preferences ‚Üí Users & Groups
  2. Select your user account
  3. Click "Login Items" tab
  4. Click "+" and add ClipPrompt.app

To disable autostart:

  1. Open System Preferences ‚Üí Users & Groups
  2. Select your user account
  3. Click "Login Items" tab
  4. Remove ClipPrompt.app from the list

Windows Instructions:

To enable autostart:

  1. Press Win + R, type "shell:startup" and press Enter
  2. Create a shortcut to ClipPrompt.exe in this folder

To disable autostart:

  1. Press Win + R, type "shell:startup" and press Enter
  2. Delete the ClipPrompt shortcut from this folder

Linux Instructions:

To enable autostart:

  1. Open your desktop environment's autostart settings
  2. Add ClipPrompt to the autostart applications list

To disable autostart:

  1. Open your desktop environment's autostart settings
  2. Remove ClipPrompt from the autostart applications list
